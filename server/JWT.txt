JWT concept: 
1. When user login the website we will assing an access token and a refresh token to that user.
2. Access token contains user's indentification(email, or role, etc) valid for shorter period of time
3. Refresh token is used to recreate assess token if that token is expired
4. If the refresh token has expired or invalid we will logout the user to login in again.
